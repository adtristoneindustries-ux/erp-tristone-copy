## üìä DATA STRUCTURE

### Hostel Schema
```javascript
{
  student: ObjectId (ref: User),
  hostelName: String,
  roomNumber: String,
  roomType: String (enum),
  roommates: Number,
  wardenName: String,
  wardenContact: String,
  wardenEmail: String,
  officeHours: String,
  laundrySchedule: [{ day, time }],
  notices: [{ title, message, date }]
}
```

### HostelIssue Schema
```javascript
{
  student: ObjectId (ref: User),
  issueType: String,
  description: String,
  priority: String (enum: Low, Medium, High, Urgent),
  status: String (enum: Pending, In Progress, Resolved, Closed),
  resolvedAt: Date,
  resolvedBy: ObjectId (ref: User)
}
```
